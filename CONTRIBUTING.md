# 🤝 Welcome to Community Shop Contributors!

Du möchtest mithelfen? Großartig! Hier ist, wie du beitragen kannst.

## 🎯 Wie Du Mithelfen Kannst

### 🐛 Bugs Melden
Findest du einen Fehler? Erstelle ein Issue mit:
- Beschreibung des Problems
- Schritten zur Reproduktion
- Erwartetes vs. tatsächliches Verhalten
- Logs oder Screenshots (wenn relevant)

### 💡 Feature-Vorschläge
Hast du eine Idee? Erstelle ein Issue und beschreibe:
- Was soll das Feature?
- Warum ist es nützlich?
- Wie würde es funktionieren?
- Gibt es Alternativen?

### 🔨 Code Contributieren

1. **Fork das Repository**
2. **Klonie den Code**
   ```bash
   git clone https://github.com/dein-username/community-shop.git
   cd community-shop
   ```
3. **Branch erstellen**
   ```bash
   git checkout -b feature/cool-new-feature
   ```
4. **Commit**
   ```bash
   git commit -m "Add: cool new feature"
   ```
5. **Pull Request**

### 📋 Code-Standards

- **PHP 8.2+**
- **PSR-12** für Code-Stil
- **Type-Hinting** überall
- **Tests** für neuen Code
- **Docs** aktualisieren

### ✅ Pull Request Checklist

- [ ] Code formatiert (PHP-CS-Fixer)
- [ ] Tests durchlaufen
- [ ] Tests für neuen Code vorhanden
- [ ] Docs aktualisiert
- [ ] keine sensitiven Daten
- [ ] commits sinnvoll benannt

### 📝 Commit-Messages

Folge dem [Conventional Commits](https://www.conventionalcommits.org/) Standard:

```
feat: new feature
fix: bug fix
docs: documentation
style: format code
refactor: code restructure
test: add tests
chore: build/ci
```

## 🚀 Schnell-Start für Contributors

1. **Repository Forken**
2. **Dependencies installieren**
   ```bash
   composer install
   ```
3. **Tests laufen lassen**
   ```bash
   phpunit
   ```
4. **Lokale Entwicklung**
   ```bash
   php -S localhost:8080 -t public
   ```

## 🎨 Design-Kontributonen

- Design-Vorschläge in Issues
- UI-Komponenten als Pull Requests
- Responsive Design testen

## 🔒 Security

Findest du ein Security Issue?
1. Erstelle ein Issue (nicht in Pull Requests)
2. Beschreibe das Problem
3. Schicke keine sensitiven Daten
4. Melde es ASAP
