## Installation

1. **Logo lokal vorbereiten:**
   ```bash
   # Logo optimieren (optional)
   # Qualität: high
   # Format: SVG (GitHub unterstützt SVG)
   ```

2. **GitHub CLI hochladen:**
   ```bash
   gh repo edit --image-path shop/github-icon.svg
   ```

3. **Oder via Webinterface:**
   - Repository-Einstellungen → Home tab → Header & Cover → Repository icon

## Best Practices

- ✅ **Format:** SVG (empfohlen) oder PNG
- ✅ **Größe:** Maximum 300x300 px
- ✅ **Hintergrund:** Transparent oder weiß
- ✅ **Design:** Einfach, skalierbar, farbenfroh
- ❌ **Vermeide:** Zu viel Detail, kleine Texte, kontrastarme Farben
