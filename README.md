## 🛍️ Community Shop - Deine Amazon Produkt-Sammlung

Ein community-gesteuerter Shop für die besten Amazon-Produkte. Bewerte mit 👍 ⭐ 🛒.

**Hinweis:** Dies ist ein **Amazon Affiliate Shop** (kein E-Commerce), da wir Amazon-Produkte anzeigen und die Community wählt. Der Checkout erfolgt direkt über Amazon.

## 🎯 Features

### ✅ MVP Features
- 🛍️ **Produkt-Katalog**: Amazon-Links automatisch extrahieren
- 🗳️ **Community Voting**: 👍 Upvote / ⭐ Favorit / 🛒 Wishlist
- 🤖 **Intelligent**: Automatisches Preis-Monitoring & Scoring
- 🔒 **Privatsphäre**: DSGVO-konform, minimale Daten
- 📝 **Score Model**: Intelligente Bewertung
- 🌐 **GitHub Pages**: Live Demo

### ⚠️ Amazon Affiliate
- ⚠️ **Kein Warenkorb**: Amazon übernimmt Checkout
- ⚠️ **Keine Payment**: Amazon Affiliate Link
- ⚠️ **Keine Shipping**: Amazon Versand
- ⚠️ **Kein Checkout**: Amazon Checkout

### 🚧 Phase 2 Features
- 📝 **Blog**: Community-Blog mit Voting
- 💬 **Forum**: Diskussionen mit Voting-Integration
- 👥 **CMS**: Content Management System
- 👤 **User-System**: Profile, Rollen, Reputation
- 🏠 **Homepage Widgets**: Top-Voted, Trending, Highlights

### 🔜 Phase 3 Features

**Hinweis:** Für Amazon Affiliate-Projekt sind Warenkorb, Payment und Shipping nicht relevant.

- ⚠️ **Warenkorb**: Nicht benötigt (Amazon übernimmt Checkout)
- ⚠️ **Payment Integration**: Nicht benötigt (Amazon Affiliate)
- ⚠️ **Order Tracking**: Amazon Order Tracking
- ⚠️ **Shipping**: Amazon Versand

**Stattdessen:**
- 📧 **Newsletter**: Community-Updates
- 🔍 **Advanced Search**: Produktfilter
- 📊 **Analytics**: Community-Stats
- 📱 **Mobile Responsive**: Mobile-first Design
- ⏳ **PWA**: Progressive Web App

## 🛠️ Tech Stack

- **Backend**: PHP 8.3+ (Laravel 11.x)
- **Frontend**: Bootstrap 5 / Vue.js
- **Datenbank**: MySQL 8.x / PostgreSQL
- **API**: RESTful JSON
- **CI/CD**: GitHub Actions
- **Docker**: Ready
