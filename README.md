## 🛠️ Technologie-Stack

- **Backend:** PHP 8.2+ (OOP)
- **Datenbank:** MySQL / MariaDB
- **Webserver:** Nginx / Apache
- **Dependency Management:** Composer

## 📦 Installation

```bash
# Repository klonen
git clone [REPO_URL]

# Abhängigkeiten installieren
cd code
composer install

# Datenbank konfigurieren
# Datenbank-Migrationen ausführen

# Webserver starten
```
