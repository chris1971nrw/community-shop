# 🛒 Dein Community-Shop für Amazon-Produkte

Entdecke die besten Produkte, die von der Community ausgewählt wurden.

## 🚀 [Produkte entdecken](#products)

---

## 🛍️ Produkt-Katalog

- Tausende von Produkten automatisch extrahiert
- Bilder, Preise, Beschreibungen
- Kategorien, Tags, Filter

## 🗳️ Community Voting

- 👍 **Upvote**: "Gefällt mir" / "Nützlich"
- ⭐ **Favorit**: "Mein Liebling"
- 🛒 **Wishlist**: "In den Warenkorb"
- 📊 **Score**: Intelligentes Ranking

## 🤖 Automatisierung

- Preis-Monitoring (täglich)
- Produkt-Erneuerung
- Entfernen veralteter Angebote
- Email-Benachrichtigungen

## 🛠️ Tech Stack

- PHP 8.3+
- Bootstrap 5
- MySQL/PostgreSQL
- REST API
- Git/GitHub
- Docker

**Wir geben die Macht zurück in die Hände der Nutzer!**

Statt algorithmischer Rankings von Unternehmen, schaffen wir eine **transparente, fair und community-getriebene Alternative**.
