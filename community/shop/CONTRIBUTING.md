# Beiträge zum Community Shop Willkommen!

Danke, dass du beiträgerst! Hier sind die Richtlinien.

## Code-Qualität

- PHP 8.2+ mit Typ-Hinweisen
- PSR-12 Coding-Style
- Kommentare für komplexe Logik
- Tests für kritische Funktionalität

## Tests

```bash
composer install
vendor/bin/phpunit
```

## Pull Requests

1. Feature-Branch aus main erstellen
2. Änderungen machen
3. Tests bestanden sein lassen
4. PR öffnen mit Beschreibung
5. Review warten

## Issue-Berichte

- Nutzt das Template im `.github/ISSUE_TEMPLATE/`
- Schreibe Reproduktionsschritte
- Füge Screenshots hinzu

## Code-Reviews

- Sei konstruktiv
- Fokus auf Qualität und Wartbarkeit
- Respektvoll bleiben

## Security

- Melde Sicherheitsprobleme über GitHub Security Advisories
- Kein Code in Issues posten

## Lizenz

Alle Beiträge unter MIT Lizenz.
