Was kann der Community Shop?

🛍️ Produkt-Katalog

Tausende von Produkten, automatisch extrahiert von Amazon mit Bild, Preis und Beschreibung.

🗳️ Community Voting

Bewerte Produkte mit 👍 ⭐ 🛒. Unser intelligentes Score-Modell zeigt die besten Produkte.

🤖 Automatisierung

Automatische Preis-Updates, Produkt-Erneuerung und Entfernen veralteter Angebote.

🔒 Datenschutz

DSGVO-konform. Minimale Datenerfassung. Transparente Privacy-Policy.

🛠️ Open Source

Vollständig in PHP geschrieben. Open Source. Beiträge sind willkommen!

🌐 API Erste

RESTful API für Entwickler. Produktdaten abrufen, Voting ausführen.

Über den Community Shop

Der Community Shop ist eine initiative, die Community-Mitglieder dabei hilft, die besten Amazon-Produkte zu finden und zu bewerten.

Statt tausende Amazon-Produkte manuell durchzusehen, sammelt die Community die besten Angebote gemeinsam.
