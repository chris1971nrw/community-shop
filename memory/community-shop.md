## 🎯 Projektidee
Community-gesteuerter, vollständig in PHP entwickelter Dynamik-Shop, bei dem User Produkte über Amazon-Links vorschlagen, abstimmen und das System automatisch profitable Produkte auswählt.

## ⚙️ Funktionsweise
1. User schlagen Amazon-Produkte vor
2. Community-Voting mit Score-Modell
3. Automatische Shop-Befüllung (Top-3 Produkte)
4. Automatische Entfernung schwacher Produkte

## 🧠 Technische Kernpunkte
- PHP-Backend (OOP/Custom Framework)
- MySQL-Datenbank
- Amazon-API-Integration
- Cron-Jobs für Shop-Optimierung
- Score-Engine für Produktrelevanz
